Programming
~~~~~~~~~~~~~

.. image:: whatyouwillneed.jpg
  :width: 100%
  :alt: Items needed to program the MSP430

You will need:
* MSP-FET.
* 4 jumper wires.
* 2x4 way 2.54mm pitch pin header.
* 1x2 way 2.54mm pitch pin header.
* 2 way jumper.
* Solder.
* cuplTag PCBA, unscrewed from the enclosure.
